🏠 House Price Prediction
---

An end-to-end Machine Learning project that predicts house prices using Linear Regression based on key features such as living area, number of bedrooms, and bathrooms.
---
📌 Project Overview

This project focuses on building a regression model to estimate house prices using real-world housing data. It demonstrates the complete machine learning workflow including:

Data preprocessing
Feature selection
Model training
Model evaluation

---

⚙️ Tech Stack
Python
Pandas
NumPy
Matplotlib
Scikit-learn

---

📊 Dataset Features Used
GrLivArea → Above ground living area
BedroomAbvGr → Number of bedrooms
FullBath → Number of full bathrooms
SalePrice → Target variable (house price)
